## Usage<a name="usage"></a>

```bash
zcrypto [OPTIONS]          # or: uv run python -m cli [OPTIONS]
```

| Option            | Description                            |
| ----------------- | -------------------------------------- |
| `-v`, `--version` | Show the application version and exit. |
| `-h`, `--help`    | Show help and exit.                    |

Running with no options prints the help.
